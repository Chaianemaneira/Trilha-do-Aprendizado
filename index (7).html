Utilização de branches

Branch é o comando git que é uma espécie de ramificação,que pode ser a principal ou main, onde é considerada o braço principal de um repositório e também tem as branches secundárias, que são o braço secundário de um repositório.

Enquanto o código de produção é mantido na principal, podemos criar outras ramificações para trabalhar avulsamente no código e depois, apenas realizarmos a junção ou o merge.

Hospedagem de Sites

Para realizar a hospedagem, é necessário um provedor de hospedagem de site para fornecer espaço em um servidor, além da armazenação de todos os arquivos e banco de dados.

Sempre que alguém digitar o domínio no navegador, o provedor transfere os arquivos necessários para atender a solicitação.
