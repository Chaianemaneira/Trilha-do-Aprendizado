Multimídias

Multimídia é uma forma de comunicação que combina diferentes formas de conteúdo como texto, imagens, aúdio e animações em vídeo em uma única apresentação.

TAG iframe.

A tag iframe é usada para incorporar outro documento ao documento HTML.
Sintaxe:

Tag vídeo.

Usada para incorporar o conteúdo de um vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo.
Contém uma ou mais tags com diferentes fontes de vídeo,onde o navegador escolhe a primeira fonte que suporta.

Tag áudio.

Usada para incorporar o conteúdo de som em um documento, como músicas ou outros fluxos de áudio.
Contém uma ou mais tags com diferentes fontes de áudio,o navegador escolhe a primeira fonte que suporta.

Tag picture.

Oferece aos desenvolvedores web mais flexibilidade na especificação de recursos da imagem.
Contém duas tags: uma ou mais tags source e uma tag img.
O navegador procurará o primeiro elemento source onde a consulta de mídia corresponde à largura da janela de visualização atual,e em seguida, mostrará a imagem adequada. O elemento img é necessário como o último filho do elemento img, como uma opção de fallback se nenhuma das tags de origem corresponder.
