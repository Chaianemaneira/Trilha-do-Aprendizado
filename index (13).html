Cascading Style Sheets

Definição:

Cascading Style Sheets, ou Folhas de Estilo em Cascata, tem por objetivo realizar a estilização do conteúdo de documentos HTML. Definindo cores, fontes, espaçamentos, alinhamentos, bordas, entre outros.

Componentes:

  • Seletores= Selecionam os elementos HTML com base no nome do elemento ao qual será aplicado o estilo.
  • Propriedades= São o aspecto específico do elemento que você deseja usar como estilo.
  • Valores= São a configuração aplicada ao atributo.

Formas de utilização:

Inline=

Os códigos CSS são definidos dentro de uma tag HTML, afetando somente a tag em que foi definido.

Incorporado ou Internal=

Os códigos de CSS são definidos no cabeçalho, dentro da tag Head, afetando somente o documento em que foi definido.

Externo ou External=

Os códigos de CSS são definidos em um arquivo externo, afetando todos os documentos que estão linkados ao CSS.
