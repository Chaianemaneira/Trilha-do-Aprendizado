Softwares de versionamento

Softwares de versionamento- Tipos

  1. Centralizado: O centralizado tem um repositório central em um servidor com controle de versões, além de ser dependentedo servidor
  2. Distribuído: O distribuído tem um repositório localcom controle de versões, e ele é independentedo servidor

Ambientes para a criação de repositórios

Um dos ambientes usados para esses meios é o software GitHub

GitHub é a plataforma que faz a hospedagem do código-fonte e arquivos com controle de versão.
