HTML

É uma linguagem de marcação de hipertexto baseada em tags, criada na década de 1990 por Tim Berners-Lee.

A HTML é usada para publicar conteúdos em hipertexto, tais como texto, imagem, áudio e vídeo,que podem ser vinculados através de hiperlinks.
  1. 1990-Criação do HTML;
  2. 1995-Versão 2.0;
  3. 1997-Versão 3.2;
  4. 1999-Versão 4.01;
  5. 2014-Versão HTML 5;

TAGS

As tags definem ao navegador a marcação de conteúdo.As tags tem funcionalidades específicas, sem o uso de tags, o navegador exibe um texto plano.
As tags podem atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.
